Maintenance of global variables | Setup routines

Detailed Description

writes MATLAB M file irootlab_setup.m

Writes several global variables into a MATLAB source file called irootlab_setup.m. This file can be later executed to restore the setup conditions.

The variables written are the graphics globals, database globals, and verbose globals.

For a list of written variables to setup, please check the source code of this file.
